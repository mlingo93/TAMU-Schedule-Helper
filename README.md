This is a chrome extension created to help students check on avability of classes.

This is a work in progress.  The intent is to allow a user to select a section and be notified when it is avaialble.

Currently, this chrome extension does the following...
1. Log into the Aggie Schedule Builder (tamu.collegescheduler.com) through the Howdy portal
2. Open the extension
3. Select the term
4. Select the subject (i.e. Physics)
5. Select the course (i.e. 207 Electricity and Magnetism for Engineering and Science)
6. Select the filters for the sections
